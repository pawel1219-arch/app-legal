Wybory dotyczące prywatności – HeadphoneEQ

Data ostatniej aktualizacji: 6 marca 2026

Twoje dane, Twoja kontrola

HeadphoneEQ nie zbiera żadnych danych osobowych, dlatego nie ma danych do zarządzania, eksportowania ani usuwania.

Dane przechowywane lokalnie

Aplikacja przechowuje ustawienia (wybrane urządzenia, presety EQ, poziom preamp) wyłącznie na Twoim urządzeniu. Aby usunąć te dane:

  • Odinstaluj aplikację HeadphoneEQ — wszystkie lokalne dane zostaną automatycznie usunięte
  • Lub zresetuj ustawienia w Preferencjach systemowych

Połączenia sieciowe

HeadphoneEQ pobiera presety EQ z publicznego repozytorium AutoEQ na GitHubie. Żadne dane użytkownika nie są wysyłane. Jeśli chcesz zablokować te połączenia, możesz używać aplikacji w trybie offline z już pobranymi presetami.

Uprawnienia systemowe

Możesz w każdej chwili cofnąć uprawnienia przyznane aplikacji HeadphoneEQ:

  • Nagrywanie audio: Ustawienia systemowe → Prywatność i bezpieczeństwo → Nagrywanie audio
  • Mikrofon: Ustawienia systemowe → Prywatność i bezpieczeństwo → Mikrofon

User Privacy Choices – HeadphoneEQ

Last updated: March 6, 2026

Your Data, Your Control

HeadphoneEQ does not collect any personal data, so there is no data to manage, export, or delete.

Locally Stored Data

The app stores settings (selected devices, EQ presets, preamp level) only on your device. To remove this data:

  • Uninstall HeadphoneEQ — all local data will be automatically removed
  • Or reset preferences via System Settings

Network Connections

HeadphoneEQ downloads EQ presets from the public AutoEQ repository on GitHub. No user data is sent. If you wish to block these connections, you can use the app offline with previously downloaded presets.

System Permissions

You can revoke permissions granted to HeadphoneEQ at any time:

  • Audio Recording: System Settings → Privacy & Security → Audio Recording
  • Microphone: System Settings → Privacy & Security → Microphone
